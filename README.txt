1)Создание контейнеров:
Находясь в корневом каталоге проекта выполнить в шеле: docker-compose build

2)Запуск проекта:
Всё там же выполнить: docker-compose up
сервис по умолчанию запускается на 80 порту, поменять это можно в ~/docker/nginx/nginx.conf

3)Создание суперюзера:
- Командой: docker ps -aqf "name=<ИМЯ_КОНТЕЙНЕРА>"
  выясняем ID контейнера web

- Командой: docker exec -it <ID_КОНТЕЙНЕРА> python manage.py createsuperuser
  попадаем в диалог создания рута

Документация: docs.txt в корневом каталоге проекта.
